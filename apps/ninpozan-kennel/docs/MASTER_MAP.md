## Notas para Agentes

- Ao criar um novo capítulo, atualizar este mapa com o status ✅
- Verificar que os links cruzados indicados realmente existem no HTML final
- Capítulos compartilhados entre pilares (ex: Linebreeding) devem ter conteúdo complementar, não duplicado
- Sempre respeitar a hierarquia de URLs definida neste mapa
- Consultar `EDITORIAL_GUIDE.md` para tom e estrutura
- Consultar `ARTICLE_TEMPLATE.md` para implementação técnica
